
This project is a simple web-based plagiarism detection application built using Flask. It allows users to upload two files and compares their textual content to determine how similar they are.

The application supports TXT, PDF, and DOCX file formats. It extracts text from each file and uses Python’s SequenceMatcher algorithm to calculate a similarity percentage between the two documents.

Based on the similarity score, the system provides a clear interpretation of the result, indicating whether the content is not plagiarized, mildly plagiarized, or highly plagiarized.

The project also includes a clean and user-friendly interface built with HTML, CSS, and Bootstrap, making it easy for users to upload files and view results instantly.

Overall, this tool demonstrates basic file handling, text processing, and web development using Flask.
